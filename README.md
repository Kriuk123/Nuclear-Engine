В планах на следующее обновление:
-Полноценные компоненты
-Сцены
-Освещение

В планах на будущее:
-Интерфейс
-Вывод 3д моделей формата obj/glTF/Fbx
-Редактор сцены

*Я программирую на любительском уровне и ничего не продаю, все мои наработки открыты для любого использования
